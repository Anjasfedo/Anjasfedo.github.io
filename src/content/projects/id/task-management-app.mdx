## Ikhtisar

Aplikasi manajemen tugas modern yang membantu tim berkolaborasi secara efektif. Buat tugas, tetapkan ke anggota tim, atur tenggat waktu, dan lacak kemajuan secara real-time.

## Fitur Utama

- **Sinkronisasi Real-time**: Perubahan disinkronkan secara instan di semua perangkat
- **Kolaborasi Tim**: Tetapkan tugas, tambahkan komentar, dan bagikan file
- **Pelacakan Kemajuan**: Bilah kemajuan visual dan papan Kanban
- **Notifikasi**: Notifikasi email dan push untuk pembaruan
- **Dukungan PWA**: Bekerja offline dan instal di desktop/mobile

## Tech Stack

- **Frontend**: Vue 3, Composition API, TypeScript
- **Backend**: Firebase (Firestore, Auth, Storage)
- **Styling**: Tailwind CSS
- **PWA**: Service Worker dengan dukungan offline
- **Manajemen State**: Pinia
