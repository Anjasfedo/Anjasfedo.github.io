## Ikhtisar

Aplikasi desktop yang dibangun dengan Electron yang membantu pengembang mengatur potongan kode mereka. Fitur penyorotan sintaks untuk 100+ bahasa, fungsi pencarian, dan sinkronisasi cloud di semua perangkat.

## Fitur Utama

- **Monaco Editor**: Editor VS Code dengan dukungan IntelliSense penuh
- **Penyorotan Sintaks**: Dukungan untuk 100+ bahasa pemrograman
- **Sinkronisasi Cloud**: Sinkronkan potongan kode di semua perangkat Anda
- **Sistem Tag**: Atur potongan kode dengan tag dan folder
- **Impor/Ekspor**: Bagikan koleksi potongan kode dengan tim Anda
- **Pencarian**: Pencarian teks lengkap di semua potongan kode

## Tech Stack

- **Framework**: Electron dengan React
- **Editor**: Monaco Editor (editor VS Code)
- **Backend**: Node.js, Express
- **Database**: SQLite untuk penyimpanan lokal
- **Cloud**: AWS S3 untuk backup/sinkronisasi
- **Pencarian**: Fuse.js untuk pencarian fuzzy
