## Overview

A modern task management application that helps teams collaborate effectively. Create tasks, assign them to team members, set deadlines, and track progress in real-time.

## Key Features

- **Real-time Sync**: Changes sync instantly across all devices
- **Team Collaboration**: Assign tasks, add comments, and share files
- **Progress Tracking**: Visual progress bars and Kanban boards
- **Notifications**: Email and push notifications for updates
- **PWA Support**: Works offline and installs on desktop/mobile

## Tech Stack

- **Frontend**: Vue 3, Composition API, TypeScript
- **Backend**: Firebase (Firestore, Auth, Storage)
- **Styling**: Tailwind CSS
- **PWA**: Service Worker with offline support
- **State Management**: Pinia
