## Overview

A desktop application built with Electron that helps developers organize their code snippets. Features syntax highlighting for 100+ languages, search functionality, and cloud sync across devices.

## Key Features

- **Monaco Editor**: VS Code's editor with full IntelliSense support
- **Syntax Highlighting**: Support for 100+ programming languages
- **Cloud Sync**: Sync snippets across all your devices
- **Tag System**: Organize snippets with tags and folders
- **Import/Export**: Share snippet collections with your team
- **Search**: Full-text search across all snippets

## Tech Stack

- **Framework**: Electron with React
- **Editor**: Monaco Editor (VS Code's editor)
- **Backend**: Node.js, Express
- **Database**: SQLite for local storage
- **Cloud**: AWS S3 for backup/sync
- **Search**: Fuse.js for fuzzy search
